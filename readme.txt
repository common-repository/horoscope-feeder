== Description ==
Horoscope Feeder is a plugin that displays your horoscope for the day, which after installation, you can use the widget or shortcode. Once enabled, you can set many options for display.

== Installation ==
1. Upload `horoscop-feeder.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin in plugin page, with a lot of settings for proper display
4. Used available widget to pun in any sidebar. Widget used settings from plugin page
5. Used shortcode [horoscope arg1=X arg2=Y...] to complet custom display in your page/post.
